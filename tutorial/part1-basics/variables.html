Go Variables

Biến dùng để lưu trữ dữ liệu.

Cú pháp

var x int = 10
y := 20

Ví dụ

package main
import "fmt"

func main() {
  x := 10
  fmt.Println(x)
}

Lỗi thường gặp

  • Dùng := ngoài function
  • Khai báo nhưng không dùng
